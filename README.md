<h2 align="center"> 🛠️ Instalación >_ 🛠️ </h2>

* $ apt update && apt upgrade -y
* $ pkg install -y git
* $ git clone https://github.com/Ux4hack/FaceDow
* $ cd FaceDow
* $ chmod +x *
* $ bash FaceDow
